banKan

banKan is a project written in Kotlin to solve people's blood needs in the most optimal way. On banKan, people can add the information they need to the announcements, and when they are no longer needed, they can remove the announcement from the "my announcements" section of their profile. You can also view donation points on Google Maps. We developed this project with 4 people. I took part in the development of the announcements and add announcement pages and designed the filter page.

mapSave

mapSave is a project developed with Kotlin language that allows you to save any place on Google Maps. This project is my first project where I use an API key. In this project where I integrated Google Maps, people can save the point they are at and access these points again later or create a route with direct navigation.
